    2. Add `Set-Alias` (per D-02) for simple commands where arguments naturally append. Examples:
       - `lg` -> `lazygit`, `lzd` -> `lazydocker`

       **CRITICAL PowerShell behavior:** `Set-Alias` can only alias to a command name, not a command with arguments. For anything with arguments (e.g., `git status`, `eza -alF`), you MUST use a `function` (per D-03).
       Group aliases neatly using `# ===` banners matching the Zsh file.
